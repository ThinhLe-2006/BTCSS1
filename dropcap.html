Bài 2.1 - CSS3 Drop Cap

HTML5 là phiên bản mới của ngôn ngữ đánh dấu HTML, cung cấp nhiều tính năng mạnh mẽ giúp xây dựng các trang web hiện đại. HTML5 hỗ trợ video, audio, canvas, các biểu mẫu thông minh và nhiều API hữu ích.

CSS3 giúp tạo hiệu ứng đẹp mắt như bo góc, đổ bóng, chuyển động, biến đổi, chia cột văn bản và nhiều khả năng trình bày giao diện chuyên nghiệp.
